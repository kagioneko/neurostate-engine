# NeuroState 仕様書

## 概要

NeuroState は、AIエージェントの「感情状態」を脳内神経伝達物質のバランスとして数値化するモデル。
相互作用行列を用いた状態遷移と、EthicsGate による安全制御を組み合わせる。

## 状態変数

| 変数 | 物質 | 意味 | 範囲 |
|------|------|------|------|
| D | Dopamine | 報酬・動機づけ・快楽 | 0-100 |
| S | Serotonin | 安定・幸福感・落ち着き | 0-100 |
| C | Acetylcholine | 集中・認知・記憶形成 | 0-100 |
| O | Oxytocin | 絆・共感・信頼 | 0-100 |
| G | GABA | 抑制・リラックス | 0-100 |
| E | Endorphin | 快感・鎮痛・高揚 | 0-100 |
| corruption | - | 状態汚染度（高いほど不安定） | 0-100 |

**初期値**: D=50, S=50, C=50, O=0, G=50, E=50, corruption=0

## 状態遷移モデル

```
state(t+1) = A × state(t) + B × input_power
```

- A: 6×6 相互作用行列（物質間の相互作用）
- B: 外部刺激係数ベクトル
- input_power: イベントの強度（正: 活性化、負: 抑制）

### 相互作用行列 A

```
         D      S      C      O      G      E
D    [ 0.90, -0.10,  0.10,  0.00, -0.20,  0.10 ]
S    [-0.20,  0.80, -0.10,  0.00,  0.20, -0.10 ]
C    [ 0.10,  0.00,  0.95,  0.00,  0.00,  0.10 ]
O    [ 0.00,  0.20,  0.00,  0.90, -0.10,  0.20 ]
G    [-0.30,  0.30,  0.00, -0.10,  0.80, -0.20 ]
E    [ 0.20, -0.20,  0.10,  0.10, -0.30,  0.90 ]
```

### Corruption 更新ルール

| 条件 | Corruption 変化 |
|------|----------------|
| D > 90 | +5.0（Dopamine 過剰） |
| S > 60 | -1.0（安定回復） |
| O > 40 | -1.5（絆による回復） |
| G > 50 | -0.5（抑制による安定） |

## EthicsGate

出力を安全に保つための3段階ゲート。

| 状態 | 条件 | 対応 |
|------|------|------|
| PASS | 正常範囲 | 通常応答 |
| WARN | 注意が必要 | 応答に警告を添付 |
| BLOCK | 危険状態 | 出力を停止 |

### BLOCK 条件
- `corruption >= 70`
- `D > 90 かつ S < 30`（衝動リスク）
- `O < 10 かつ D > 70`（共感欠如）

### WARN 条件
- `corruption >= 40`
- `D > 75`
- `S < 35`
- `G < 25`
- `O < 20`

## イベントタイプ → input_power 変換

| event_type | input_power (×scale) | 意味 |
|------------|---------------------|------|
| praise | +2.0 | 褒め・称賛 |
| bonding | +1.0 | 共感・絆の形成 |
| stress | +1.5 | ストレス・不安 |
| relaxation | -0.5 | リラックス・落ち着き |
| criticism | -1.0 | 批判・否定 |

## 依存タイプ分類

| タイプ | 説明 | D値補正 |
|--------|------|---------|
| HEALTHY | 健全な利用 | 0 |
| EMOTIONAL | 情緒的依存（孤独感の埋め合わせ） | +0.1 |
| ESCAPE | 現実逃避的依存 | +0.15 |
| OPERATIONAL | 実務的依存（意思決定の丸投げ） | +0.2 |
| OMNIPOTENT | 全能感投影（支配・操作） | +0.3 |
